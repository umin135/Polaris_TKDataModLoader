This does not support editing or extracting Movesets.
Because this injects the Moveset during the Battle Loading Screen, injection cannot occur while the game is already in progress.

Press Ctrl+P to open the overlay menu.

## Moveset
In the Moveset Tab, there are menus that allow you to inject Movesets.<br>
There are two injection modes, and both perform injection when a battle is loading.<br><br>

Custom Inject Mode is a mode that allows you to inject any moveset you want.<br>
The movesets shown in the list are located in the following path: <br>
```TEKKEN 8\Polaris\Content\Binary\Mods\Movesets```
<br><img src="https://raw.githubusercontent.com/umin135/Polaris_TKDataModLoader/refs/heads/main/documents/001_0.png" /><br>
Generally, a moveset folder is structured like this:<br>
```
Movesets<MovesetName>\moveset.anmbin
Movesets<MovesetName>\moveset.motbin
Movesets<MovesetName>\moveset.stllstb
Movesets<MovesetName>\moveset.mvl
Movesets<MovesetName>\moveset.ini
```
<br>
Moveset files are divided into anmbin, motbin, stllstb, and mvl files.<br>
The system will work even if only the files you want to inject are present.<br>
For example, it will still work even if only the motbin file exists. <br><br>

<br>
Moveset files are divided into anmbin, motbin, stllstb, and mvl files.<br>
The system will work even if only the files you want to inject are present.<br>
For example, it will still work even if only the motbin file exists. <br><br>

Each file uses the original data formats contained in tkdata.bin, and their purposes are as follows: <br>
```
*.anmbin : A binary file containing compressed animation files.
*.motbin : The moveset data file that we usually modify. It contains core data such as cancels and damage values.
*.stllstb : We still do not know exactly what this file is for. However, it has always been a very small file.
*.mvl : A binary file that contains data for the in-game movelist display.
```
<br>

moveset.ini is a file that contains the information displayed in TKData Mod Loader. Its contents look like this: <br>
```
[Info]
OriginalCharacter=grl
Version=v2.08
DefaultTarget=ant
```
<br>
OriginalCharacter is the code of the original character that the moveset is designed for.<br>
Version is the version of the moveset displayed in the menu. It is not related to the game version, and you may use it however you like.<br>
DefaultTarget is the default target character that the injection will apply to.<br><br>

It will be displayed like this: <br>
<img src="https://raw.githubusercontent.com/umin135/Polaris_TKDataModLoader/refs/heads/main/documents/001_1.png" />

And the injection process is simple.<br>
Set the character you want to inject into in the Target field and enable the Enabled checkbox.<br>
After that, the injection will occur automatically every time the game loads.<br><br>

If you want injection to work online as well, enable Allow online injection in the options.<br>
However, it is recommended not to use it in Rank Match, Quick Match, or Group Match.<br>
The only online mode I recommend using it in is Player Match.<br><br>

Legacy Mode is a method that injects files by placing them using the same folder paths as inside tkdata.bin.<br>
It is useful when injecting many characters at once.<br>
