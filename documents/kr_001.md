# 기본 사용 방법
이것은 Moveset을 편집하거나 Extract하는 기능을 지원하지 않습니다.
이것은 Battle Loading Screen에서 Moveset을 인젝트하기 때문에 게임이 진행중일때는 인젝트를 할 수 없습니다.

Ctrl+P를 누르면 오버레이 메뉴가 나타납니다.

fbsdata Tab은 아직 완성되지 않았기때문에 작동하지 않습니다.
fbsdata는 일반적으로 게임이 실행될때 오직 1번 Load하는 캐릭터 리스트, 스테이지 리스트, 아이템 목록에 대한 것입니다.

Moveset Tab에서는 Moveset을 인젝트할 수 있는 메뉴들이 있습니다.

인젝트 모드는 총 2가지로 모두 배틀이 로딩될때 인젝트가 발생합니다.

Custom Inject Mode는 원하는 moveset을 인젝트할 수 있는 모드입니다.
리스트에 표시되는 무브셋들은 다음경로에 있는 무브셋들입니다 : 
TEKKEN 8\Polaris\Content\Binary\Mods\Movesets
<img src="https://raw.githubusercontent.com/umin135/Polaris_TKDataModLoader/refs/heads/main/documents/001_0.png" />
일반적으로 무브셋폴더는 다음과 같이 구성됩니다.
Movesets\<MovesetName>\moveset.anmbin
Movesets\<MovesetName>\moveset.motbin
Movesets\<MovesetName>\moveset.stllstb
Movesets\<MovesetName>\moveset.mvl
Movesets\<MovesetName>\moveset.ini

무브셋 파일은 anmbin, motbin, stllstb, mvl파일로 나뉩니다.
이중에서 인젝트하길 원하는 파일만 존재해도 작동합니다.

각 파일은 tkdata.bin에 포함된 실제 원본 데이터 형식이며, 용도는 다음과 같습니다 : 
*.anmbin : 애니메이션 파일들이 압축된 바이너리 파일입니다.
*.motbin : 우리가 일반적으로 수정했었던 무브셋 데이터파일입니다. 캔슬/대미지등의 핵심적인 데이터들이 포함되어있습니다.
*.stllstb : 아직 우리는 이것이 실제로 무엇인지 파악하지 못했습니다. 하지만 항상 매우 작은 크기의 파일이었습니다.
*.mvl : 게임내에서 표시할 무브리스트에 대한 데이터가 포함되어있는 바이너리 파일입니다. 

moveset.ini는 TKData Mod Loader에 표시한 정보를 담은 파일입니다. 내용물은 다음과 같습니다 : 
```
[Info]
OriginalCharacter=grl
Version=v2.08
DefaultTarget=ant
```
OriginalCharacter는 무브셋에 딱 맞는 원본 캐릭터의 코드입니다.
Version은 메뉴에서 표시한 무브셋의 버전입니다. 게임버전과는 연관이 없지만 용도는 원하시는대로 활용하시면 됩니다.
DefaultTarget은 인젝트할 대상 캐릭터의 기존값입니다.
